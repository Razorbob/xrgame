## Installation

### Step 1
```
git clone https://github.com/Razorbob/xrgame.git
```
### Step 2
```
cd xrgame
```
### Step 3
```
npm install
```
### Step 4
```
npm run start
```
### Step 5
connect with Browser to https://Your_PC_IP:3000
